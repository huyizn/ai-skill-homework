# AI 應用作業：使用 AI Skill 產生網頁檔案

## 一、 使用的 AI Skill 說明
本作業在 AI 系統中建立並使用了一個名為 **「輕霧藍紫流派-響應式網頁前端開發專家」** 的 Skill。

* **Skill 核心功能**：接收使用者的文字簡介，自動轉化為具備現代極簡感、莫蘭迪低飽和度（霧霾藍與薰衣草淺紫）色系、且支援手機瀏覽的單頁式 HTML 網頁。
* **指令設計**：
    1. 限定輸出為單一 `.html` 檔案（內嵌 CSS 樣式）。
    2. 配色限制使用低飽和度淺藍（#a3b8cc）與淺紫（#c5bfe0），呈現柔和、清透的視覺審美。
    3. 限制不調用外部圖片，大頭貼與裝飾皆以 CSS 幾何圖形與漸層色呈現。

## 二、 執行過程與成果
透過啟動該 Skill，輸入個人基本資料與審美偏好後，AI 成功精準地輸出符合格式的 HTML 程式碼。網頁視覺色彩和諧，完全符合預期設計。
